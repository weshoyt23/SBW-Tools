How It Works
01
The Basics
▼

Midjourney is a text-to-image AI. You type a description — called a prompt — and it generates four image variations. You then upscale the ones you like, generate new variations, or refine with follow-up prompts.

It runs inside Discord. You type your prompts in a channel, the bot responds with your images right there in the chat. It feels awkward at first and second nature by week two.

The command to generate is always /imagine followed by your prompt. Everything else is refinement.

02
The Workflow
▼
1
Type /imagine + your prompt in Discord
↓
2
MJ generates 4 variations — labeled U1–U4 (upscale) and V1–V4 (variation)
↓
3
Hit U on the best one to upscale to full resolution
↓
4
Hit V to generate 4 more variations of that specific image
↓
5
Refine prompt and repeat until you get what you need

Pro move: Add --seed [number] to lock in a composition you love, then tweak the prompt around it.

03
Prompt Anatomy
▼

A well-structured prompt layers information in a specific order. Each layer narrows the output toward your vision. Missing layers = unpredictable results.

What is in the image

Style

Medium
How it was "made"
shot on medium format
Lighting
The light source + quality
dramatic side lighting, high contrast
Mood
The feeling
muted tones, minimal composition

04
Key Parameters
▼
--ar 9:16
Aspect ratio. 9:16 for Reels, 1:1 for feed posts, 4:5 for portrait feed, 16:9 for banners. Always set this.
--style raw
Turns off MJ beautification. Use when you want gritty, photorealistic, or unpolished results. Essential for SBW's aesthetic.
--v 7
Model version. Always use the latest. V7 is current as of 2026. Better detail, lighting, and coherence.
--no [thing]
Exclusion. --no text, watermarks, logos, people is almost always worth adding to product shots.
--sref [URL]
Style reference. Drop in an image URL and MJ matches its visual style. This is where your inspiration folder becomes a superpower.
--cref [URL]
Character reference. Keeps a face or figure consistent across multiple generations. Useful for a recurring model or character.
--seed [number]
Locks composition. Find a layout you love, get the seed number, then iterate the prompt around that fixed composition.
--chaos [0–100]
Variation range. Higher = more unpredictable and experimental results. Lower = more consistent and predictable.
--stylize [0–1000]
MJ's artistic influence. Lower = closer to your prompt. Higher = MJ adds more of its own aesthetic. Default is 100.
05
Pro Tips
▼
Tip 01 — Don't accept generation 1
A professional treats MJ as a conversation, not a vending machine. Generate, identify what's almost right, isolate the specific element that isn't working, fix just that thing in the prompt. Generation 8 of the same prompt is often the keeper.
Tip 02 — Build a prompt library
Every prompt that produces something great gets saved with notes on what worked. Over time you build a personal dictionary of language that reliably produces the SBW aesthetic. That's what the Library tab is for.
Tip 03 — Use your inspiration folder as sref
When you save an image to your Inspiration tab, you're building style reference material. Upload those images anywhere publicly accessible and use the URL with --sref for dramatically more targeted results.
Tip 04 — Describe visuals in words
When you see a photo you love, stop and articulate it. Is it the lighting? The grain? The color temperature? The composition? Training yourself to describe visuals in words is the core skill MJ rewards. This is what you're doing when you log inspiration.
Tip 05 — Develop SBW base prompts
Build a set of base prompts for each SBW content type — product shots, lifestyle imagery, brand mood pieces — and swap out specific details while keeping the core style language constant. That's how you build a visually coherent brand across hundreds of images.
Tip 06 — Browse the community showcase first
Before you ever subscribe, go to Midjourney's website and browse the public showcase. When you see an image you love, click it and read the prompt. 20 minutes a day and you'll absorb prompt language faster than any tutorial.
Tip 07 — Relax Mode for volume
Standard plan and above includes unlimited Relax Mode generations — slower but free. Use Fast Mode for when you're iterating in real time, Relax Mode for batch exploration when you're not in a hurry.
06
SBW-Specific Strategy
▼

Your unfair advantage is that you already know what SBW looks and feels like. MJ can't define that — you define it. MJ executes it at scale.

Content Types to Build Base Prompts For
Product Shots
Flat lay, hanging, worn — consistent lighting, consistent surface
Lifestyle
Worn in Austin — streets, architecture, natural light, real context
Mood Boards
Textures, color palettes, abstract reference material for drops
Design Reference
Graphic motifs, typography directions, pattern research
Drop Teasers
Atmospheric, partially obscured, cinematic — build anticipation

The learning curve is real but short. The first week feels frustrating. Most designers get genuinely good within two to three weeks of daily use. Start before you have MJ access. The preparation work — building your inspiration library and learning to describe visuals — is the actual skill.
